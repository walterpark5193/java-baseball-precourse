## 구현할 기능 목록
1. 랜덤 3자리 숫자 생성 기능
   - 같은 숫자는 1번만 나오도록 구현

2. 사용자 정답값 입력 받는 기능
   - 3자리 숫자가 아니면 에러 발생 (자릿수, 숫자여부 확인)

3. 입력된 숫자가 정답에 존재하는지 체크하는 기능
   - input : 한자리 숫자
   - output : YN

4. 같은 자릿수에 동일한 숫자가 존재하는지 체크하는 기능
   - input : 한자리 숫자
   - output : YN

5. 스트라이크, 볼 카운트 체크하는 기능
   - input : 숫자 리스트
   - output : N/A
   
6. 정답을 맞췄을 시 게임을 새로 시작하거나 종료하는 기능 
   - 새로시작-1, 종료-2 가 아니면 오류 발생
